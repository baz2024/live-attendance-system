# Live Attendance System (Frontend + API)

This repository contains two parts:

- **API**: Node.js + Express + MySQL + JWT
- **Frontend**: React + Vite + Material UI

## Quick start (recommended order)

### 1) Start the API
1. Open `README_API.md`
2. Create the API `.env`
3. Install + run:

```bash
npm install
npm run dev
```

Test:

```bash
curl http://localhost:5050/health
```

### 2) Start the Frontend
1. Open `README_FRONTEND.md`
2. Create the frontend `.env`
3. Install + run:

```bash
npm install
npm run dev
```

Open the app:

- http://localhost:5173
